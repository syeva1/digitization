# Основы лидерства

Курс Сколково (Leadership Fundamentals): лидерство, власть, этика, команда, критическое мышление.

## Для кого и зачем

Курс даёт основы лидерства: власть, миссия, стили лидерства, этика, команда и групповая динамика, критическое мышление, кейс «Эверест»; материалы «The Drawbridge». Подходит для развития лидерских компетенций в рамках BBA.

## Содержание

Темы: описание курса (Leadership Fundamentals Syllabus 2023); лидерство и власть (Шапенко); миссия, стили, этика; команда; групповая динамика; критическое мышление; кейс «Эверест»; литература; презентации; вводный блок; «The Drawbridge» (рус.).

## Как устроены материалы

- **Силлабус** — программа курса (Leadership Fundamentals Syllabus 2023).
- **Введение_лидерство** — вводный блок (ранее «День 1»).
- **Литература** — литература к курсу (ранее «Литература к курсу… копия»).
- **Презентации_курса** — презентации к курсу (при наличии после переименования).
- Тексты в корне: **Лидерство_Шапенко_BBA**, **Миссия_Шапенко_BBA**, **Стили лидерства_Шапенко_BBA**, **Ethics_Shapenko_BBA**, **Power_Shapenko_BBA**, **Команда**, **Групповая динамика BBA**, **Critical Thinking_Shapenko_ВВА**, **Culture in Leadership_BBA**, **Everest_Shapenko_BBA**.
- **The Drawbridge_Rus** — материалы кейса «The Drawbridge» (рус.).

Рекомендуемый порядок: силлабус → Введение_лидерство → темы по программе (власть, миссия, этика, команда, групповая динамика, критическое мышление, Эверест, Drawbridge).

## Язык и программа
