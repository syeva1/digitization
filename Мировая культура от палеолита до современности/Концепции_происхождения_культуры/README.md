# Концепции происхождения культуры

Классические тексты по концепциям происхождения культуры: Кассирер, Фрейд, Фрезер, Хейзинга, Пропп, библейские тексты и др.

## Содержимое

Опыт о человеке (Кассирер), Недовольство культурой (Фрейд), Русские аграрные праздники (Пропп), Человек играющий (Хейзинга), Книга Исход и др. Используется в начале курса для теоретической базы.
