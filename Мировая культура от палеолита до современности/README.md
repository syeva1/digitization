# Мировая культура от палеолита до современности

Курс Сколково: история мировой культуры, античность, Византия, Древняя Русь, театр, искусство Греции и Рима.

## Для кого и зачем

Курс даёт обзор мировой культуры от палеолита до современности: концепции происхождения культуры, античная культура, искусство Древней Греции и Рима, Византия и Древняя Русь, лекции М.И. Свидерской, театр, Виппер (введение). Подходит для интереса к истории культуры без обязательной предподготовки.

## Содержание

Темы: силлабус (Мировая культура 2024, итог); концепции происхождения культуры; античная культура; искусство древней Греции (физтех); Византия и Древняя Русь; лекции Свидерской; материалы по лекциям 27.05, 28.05, 30.05, 31.05; театр; А. Большаков «Человек и его двойник»; 31_Wipper (введение).
