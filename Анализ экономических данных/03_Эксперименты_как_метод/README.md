# Литература: эксперименты в экономике

Тема: статьи по экспериментальной экономике (Smith, Chamberlin, Isaac-Walker, McCarter и др.).

## О чём материалы

Классические и современные работы по методам и результатам экономических экспериментов. Используются в блоке курса, посвящённом экспериментам.

## Связь с курсом

Соответствует тематическому блоку по экспериментам; дополняет лекции и блок «Эвристики_и_искажения».
