# Дополнительные лекции

Тема: дополнительные лекционные материалы курса «Анализ экономических данных».

## О чём материалы

Лекции и слайды, не входящие в блок «Введение_анализ_данных»; темы уточняются по именам файлов в папке.

## Связь с курсом

Дополняют основные блоки; порядок использования задаётся силлабусом.
