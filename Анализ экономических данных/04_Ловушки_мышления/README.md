# Эвристики и когнитивные искажения

Тема: эвристики (репрезентативность, доступность, якорение) и когнитивные искажения (endowment effect, confirmation bias, self-serving bias, decoy effect и др.).

## О чём материалы

Тексты и выжимки по поведенческой экономике и психологии принятия решений: как люди упрощают задачи (эвристики) и какие систематические ошибки допускают (искажения).

Один из основных тематических блоков; связан с «Литература_эксперименты» и «Литература_к_курсу».
