# Анализ экономических данных

Курс (BBA): методы анализа данных, эксперименты, эвристики и искажения, NLP.

## Для кого и зачем

Курс даёт практические навыки анализа экономических данных: введение в методы, эксперименты (Smith, Chamberlin, Isaac-Walker, McCarter и др.), эвристики и когнитивные искажения (якорение, endowment effect, репрезентативность и т.д.), работа с LMS, текстовые данные (NLP). Подходит для студентов BBA с интересом к данным и экспериментам.

## Содержание

Темы: введение в анализ данных (лекция Паршакова); эксперименты и методология (статьи и обзоры); эвристики и искажения; LMS-платформы; NLP и текстовые данные; дополнительная литература и лекции.

## Как устроены материалы

- **Силлабус** — программа курса (Анализ экономических данных BBA 2025–26).
- **Введение_анализ_данных** — первая лекция и вводные материалы.
- **Литература_эксперименты** — статьи по экспериментальной экономике (Smith, Chamberlin, Isaac-Walker, McCarter и др.).
- **Эвристики_и_искажения** — материалы по эвристикам и когнитивным искажениям (якорение, endowment, репрезентативность, confirmation bias и т.д.).
- **Литература_к_курсу** — общая литература (Behavioral Economics, Game Theory, Methods in Experimental Economics).
- **Тема_LMS_платформы** — материалы по работе с LMS.
- **Тема_NLP_текстовые_данные** — материалы по обработке текстов.
- **Лекции_дополнительные** — дополнительные лекции курса.

Рекомендуемый порядок: силлабус → Введение_анализ_данных → Литература_эксперименты и Эвристики_и_искажения → остальные блоки по интересу.

## Язык и программа

Материалы на английском и русском. Курс: BBA.
