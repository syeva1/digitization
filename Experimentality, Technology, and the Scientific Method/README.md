# Experimentality, Technology, and the Scientific Method

Курс Сколково (Year 1, Module 1). Преподаватель: Siyaves Azeri, PhD.

## Для кого и зачем

Курс даёт базовые концепции философии науки: в чём специфика научного знания, как устроены научные теории и методы, как меняется наука. Полезен для понимания споров о выборе теорий, роли теории в науке и научном прогрессе. Уровень — введение в философию науки, предварительная подготовка по философии не обязательна.

## Содержание

Основные темы: индукция и проблема демаркации науки, фальсификационизм Поппера, методология исследовательских программ Лакатоса, научные революции (Кун, Фейерабенд), естественная эпистемология и онтологическая относительность Куайна. Авторы: Карл Поппер, Имре Лакатос, Томас Кун, Пол Фейерабенд, Уиллард Куайн, Дэвид Юм.

## Как устроены материалы

- **Силлабус** — папка `Силлабус/` с программой курса (Syllabus 2023): описание, расписание тем, критерии оценивания.
- **Введение_индукция_фальсификационизм_Поппер** — вводный блок: индукция, верификационизм, фальсификационизм Поппера; текст «Логика научного открытия».
- **Куайн_эпистемология_онтология** — блок по Куайну: эпистемология, «С точки зрения логики», онтологическая относительность.
- **Литература_на_русском** — переводы и тексты на русском (Поппер, Кун, Лакатос, Фейерабенд, Куайн, Юм).

Рекомендуемый порядок: сначала силлабус, затем блок «Введение_индукция_фальсификационизм_Поппер», далее блок «Куайн_эпистемология_онтология»; литературу на русском можно использовать параллельно по темам.
