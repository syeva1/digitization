# Введение: индукция, фальсификационизм (Поппер)

Тема блока: научный метод, проблема индукции, верификационизм и фальсификационизм Карла Поппера.

## О чём материалы

Базовый блок курса. Рассматриваются: «научный метод», наивный индуктивизм, проблема индукции, проблема теории и наблюдения, критика Поппером марксизма и психоанализа, решение Поппера проблемы индукции, фальсификационизм и его ограничения. Основной текст — «Логика научного открытия» (Popper, The Logic of Scientific Discovery).

## Связь с курсом

Открывает курс; без этого блока сложнее понимать Лакатоса (критика фальсификационизма) и последующие темы. После него идут блоки по Лакатосу/Куну/Фейерабенду и по Куайну.

## Основные файлы

- `karl-popper-the-logic-of-scientific-discovery-1.txt` — оцифрованный текст книги Поппера (фрагменты по программе).
