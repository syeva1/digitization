# Академическое письмо

Курс Сколково: основы академического письма и аргументации.

## Для кого и зачем

Курс даёт навыки академического письма: структура текста, аргументация, работа с источниками. Подходит для тех, кто пишет эссе, курсовые или исследовательские тексты в рамках программы. Предварительная подготовка не обязательна.

## Содержание

Темы: введение в академическое письмо (форматы, требования), аргументация (построение аргументов, критика источников). Материалы для работы автономно и с тьютором/ОС.

## Как устроены материалы

- **Силлабус** — программа курса на русском (файлы в папке).
- **Силлабус_англ** — программа на английском (Academic Writing Syllabus 2023).
- **Введение_академическое_письмо** — материалы первого блока: введение, материалы для автономной группы и для группы с ОС.
- **Аргументация** — материалы по теме аргументации (презентация и текст).

Рекомендуемый порядок: силлабус, затем блок «Введение_академическое_письмо», затем «Аргументация».

## Язык и программа

Материалы на русском и английском. Курс входит в программу BBA.
