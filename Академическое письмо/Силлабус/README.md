# Силлабус (рус.)

Тема: программа курса «Академическое письмо» на русском языке.

## О чём материалы

Официальный силлабус курса: описание, цели, содержание, требования. Файлы в папке — текст программы и возможные дополнительные форматы.

## Связь с курсом

Имеет смысл прочитать в начале вместе с блоком «Силлабус_англ» для полной картины курса.
