# Коммуникация с данными

Курс: представление данных, визуализация, ИИ в коммуникации.

## Для кого и зачем

Курс учит представлять данные понятно: визуализация, сторителлинг, использование ИИ в коммуникации с данными; тема поколенческого разрыва. Подходит для тех, кто работает с отчётами и презентациями на основе данных.

## Содержание

Темы: программа курса (силлабус 2); ИИ в контексте коммуникации с данными (BBA); поколенческий разрыв (коммуникация между поколениями).

## Как устроены материалы

- **Силлабус** — программа курса «Коммуникация с данными» (силлабус 2).
- **Тема_ИИ_в_коммуникации** — материалы по ИИ в курсе (17.09.2025, BBA).
- **Поколенческий разрыв** — материалы по теме поколенческого разрыва.

Рекомендуемый порядок: силлабус → Тема_ИИ_в_коммуникации → Поколенческий разрыв (или по программе).

## Язык и программа

Материалы на русском и/или английском. Курс: BBA.
