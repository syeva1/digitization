# Проектный менеджмент

Курс Сколково (BBA): управление проектами, приёмка, приживление, план по аспектам.

## Для кого и зачем

Курс даёт основы проектного менеджмента: блоки программы (BB2 УП), план приёмки и приживления, сводный план по аспектам, чеклист, шаблон описания (двухстраничный). Подходит для BBA и применения в проектной работе.

## Содержание

Темы задаются силлабусом (Силлабус_Проектный_менеджмент_2025): вводные блоки (BB2 УП, Д1–3), часть 4, части 5–6; план приёмки и приживления; сводный план по аспектам; чеклист; шаблон описания.

## Как устроены материалы

- **Силлабус_Проектный_менеджмент_2025** — программа курса.
- **BB2 УП**, **BB2 УП Д1-3**, **BB2 УП Часть 4**, **BB2 УП Часть 5-6** — блоки курса по программе.
- **План_приемки_и_приживления**, **Сводный план по аспектам**, **Чеклист_Сводный_план_по_аспектам**, **Шаблон Описания двухстраничный** — шаблоны и планы.

Рекомендуемый порядок: силлабус → блоки BB2 УП по порядку → планы и шаблоны по заданию.

## Язык и программа

Материалы на русском. Курс: BBA.
