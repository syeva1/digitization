# Проектный менеджмент

Курс Сколково (BBA): управление проектами, приёмка, приживление, план по аспектам.

**Полный указатель материалов:** [INDEX.md](INDEX.md)

## Для кого и зачем

Курс даёт основы проектного менеджмента: программа курса, учебник (Настольная книга слоновщика), лекции по дням (1, 1–3, 4, 5–6), план приёмки и приживления, сводный план по аспектам, чеклист, шаблон описания (двухстраничный). Подходит для BBA и применения в проектной работе.

3. **Лекции по дням** — 02_Лекции_день_1_введение, 03_Лекции_дни_1-3, 04_Лекции_день_4, 05_Лекции_дни_5-6.
