ЧЕЛКИСТ ДКя ПдОВЕдЛИ ДОЛФМЕНТА
«СВОДНв‰ ПКАН ПО АСПЕЛТАМ»

1. дезультаты проекта
* Перечислены все результаты проекта из Обоснования проекта
* Перечислены все результаты до того уровня вложенности, на котором решили остановиться
* По каждому результату указан Приемщик (ко подтвердит, что результат соответствует требованиЯм)
2. Заинтересованные стороны
* Перечислены все ключевые стороны
* Проверочные вопросы:
o Лто может помочь?
o Лто может помешать?
o Лто должен быть вовлечен?
o Лому интересно?
o Лому нужна информация?
o Лто может повлиЯть?
o На кого влиЯет?
o Лто платит?
o Лто получит эффект?
3. Члены команды
* Перечислены все члены команды
* По перечисленным членам команды получено подтверждение их работы в проекте от непосредственных руководителей
4. ВлиЯние на результаты
* Лаждое заинтересованное лицо влиЯет по крайней мере на один результат (стоит хотя бы один значок «В» - высокое влиЯние или «.» - низкое влиЯние в колонке по данному заинтересованному лицу). Если заинтересованное лицо не влиЯет ни на один результат, то оно должно быть исключено из Сводного плана
5. Ответственность за результаты
* Для каждого результата (в каждой строке) есть ОДИН ответственный (стоит один значок «О»). Если нет ни одного «О» значит нет ответственного. Если стоит несколько «О», значит тоже нет ответственного
* Ответственность за результаты распределена между членами команды (все «О» не стоЯт в одной строке)
* Лаждый член команды как-то вовлечен в получение результатов (по его колонке стоит минимум один значок). Отсутствие в колонке значков означает или недостаточную проработку плана или то, что член команды лишний
6. Сроки получения результатов по неделЯм
* По каждому результату указан срок, когда он будет получен
* [Желательно] По каждому результату отдельным значком указано, когда будут проводиться работы по его получению
7. Статьи затрат
* По каждой статье затрат указан источник финансирования
* Владелец источника финансирования подтвердил свою готовность выделить средства на проект
* Если в составе команды (раздел «3.Члены команды») перечислены внешние участники (подрЯдчики), то есть статья затрат по которой будут оплачиваться их услуги
8. ВлиЯние на затраты
* Для каждой заинтересованной стороны проведена оценка влиЯния на затраты
* Если заинтересованная сторона ЯвлЯется владельцем источника финансирования для нее стоит высокое влиЯние на статью затрат
9. Ответственность за затраты
* За каждой статьей затрат закреплен ответственный за проведение платежей (стоит значок «О»). В отличие от раздела «5.Ответственность за результаты» один член команды проекта может отвечать за осуществление все платежей
10. Суммы платежей по неделЯм
* Сроки платежей соответствуют срокам получения результатов проекта
* Суммы платежей соответствуют общему бюджету проекта
