# Прикладная социология и социологические методы

Курс Сколково: теории порядка и действия, концептуализация, методы исследований, анализ данных, Бейтсон.

## Для кого и зачем

Курс даёт прикладные и методологические основы социологии: теории порядка, теории действия, концептуализация, методы исследований, анализ данных и интерпретация; текст Бейтсона «Теория игры и фантазии». Подходит для углубления после «Fundamentals of Sociology».

## Содержание

Темы по программе (Прикладная социология 2024, финал): блоки 1–5 (теории порядка, действия, концептуализация, методы, анализ данных); ридинги к занятиям 17.06 и 18.06; Бейтсон «Теория игры и фантазии».

## Как устроены материалы

- В корне: **Прикладная социология (2024) (ФИНАЛ).txt** — программа; **Блок 1–5** — тексты блоков; **Бейтсон_Теория игры и фантазии.txt**.
- **Ридинг_темы_17_06** — тексты для чтения к темам 17.06.
- **Ридинг_темы_18_06** — тексты для чтения к темам 18.06.

Рекомендуемый порядок: программа → Блок 1–5 по порядку → ридинги по датам; Бейтсон по программе.
