# Microeconomics

Курс Сколково (BBA 1, Module 5). Преподаватель: Shahir Safi, PhD.

## Для кого и зачем

Курс учит «думать как экономист»: моделирование решений агентов (индивиды, домохозяйства, фирмы), предсказания на микро- и макроуровне. Темы: социальный выбор и благосостояние, рыночная экономика, несовершенная конкуренция, информация и коммуникация, неформальные рынки (сети). Подходит для первого знакомства с микроэкономикой.

## Содержание

Темы по программе: введение и социальный выбор (предпочтения, агрегация, благосостояние); рыночная экономика; несовершенная конкуренция; несовершенная информация; коммуникация; неформальные рынки и социальные сети.

## Как устроены материалы

- **Силлабус** — папка `Силлабус/`: программа курса (Microeconomics Syllabus 2025).
- **Тема_Социальный_выбор_и_благосостояние** — лекция и материалы по теме 1: предпочтения, агрегация, эффективность и справедливость.
- **Сети_и_экономика** — материалы по сетям и связанным темам (по программе).

Рекомендуемый порядок: силлабус, затем блок по социальному выбору, далее остальные темы по программе.
