# Сети и экономика

Тема: материалы по сетям и связанным темам микроэкономики (по программе курса).

## О чём материалы

Дополнительные материалы курса микроэкономики: неформальные рынки, социальные сети, приложения теории сетей в экономике (согласно силлабусу — Topic по сетям).

## Связь с курсом

Соответствует блоку программы про informal markets и social networks; опирается на понятия из тем про предпочтения и рынки.

## Основные файлы

Файлы в папке — тексты и материалы по теме сетей (имя файла может содержать «cnet» или аналогичное обозначение).
