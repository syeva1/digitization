# Библиотека курсов

Материалы курсов бакалавриата и магистратуры, оцифрованные из презентаций и книг. Структура и названия приведены к тематическим (без привязки к дням); в каждой папке есть README с описанием. Курсы сделаны максимально автономными — по тексту и README можно разобраться без прохождения курса.

## С чего начать

1. Выберите курс в таблице ниже и откройте папку курса.
2. В папке курса откройте **README.md** — там описание курса, содержание и порядок модулей (Модуль 1, 2, 3… или нумерованные папки 01_, 02_…).
3. Следуйте нумерации: сначала первый модуль/блок, затем следующий. В README каждой подпапки указано, что внутри и что читать или делать.
4. В начале каждого файла .txt есть блок **Тема / Источник / Содержание** — по нему видно, к какой теме относится файл и откуда материал.
5. Если в курсе есть блок с книгами или ридингами — в README этой папки написано, **зачем каждая книга** и как она помогает в теме (что закрывает, какие навыки даёт).

## Система файлов и порядок прохождения

**Уровень 1 — репозиторий:** в корне лежат папки курсов (по одной папке на курс). Названия курсов — по смыслу, без дат (например: «Проектный менеджмент», «Fundamentals of Sociology»).

**Уровень 2 — внутри курса:** модули (разделы). Что проходится **первым**, что **вторым**, задаётся так:
- **Первое** — всегда программа курса (силлабус): папка типа `Силлабус/` или `01_Программа_курса_...`; в README курса это «Модуль 1» или «Начните с неё».
- **Второе** — введение или базовый теоретический блок (например `02_Лекции_день_1_введение`, «Модуль 2. Введение»).
- **Третье и далее** — остальные темы по порядку: нумерованные папки `03_...`, `04_...` или в README «Модуль 3», «Модуль 4» с перечислением папок и файлов.

Названия папок — тематические, без дат в имени; где важен порядок — префикс `01_`, `02_`, … или явная нумерация в README.

**Уровень 3 — внутри модуля:** файлы .txt (и при необходимости .md). Названия файлов — по теме или по автору/названию материала (без « — копия», без лишних суффиксов вроде «c», «л»). В начале каждого .txt — блок **Тема / Источник / Содержание**, чтобы по одному файлу было понятно, что это и зачем.

**Итог:** заходите в курс → открываете README → читаете «Модуль 1, 2, 3…» или список папок 01_, 02_, … — по этому порядку и проходите материалы; в каждой папке README подсказывает, что внутри и что делать.

## Как пользоваться

- Зайдите в папку нужного курса и откройте **README.md** — там описание курса, для кого он, содержание и навигация по подпапкам (модулям).
- В каждой подпапке тоже есть **README.md** с темой блока и кратким описанием материалов; у папок с книгами/ридингами — пояснение **зачем эта книга**, как она помогает в курсе.
- Названия папок и файлов — по **темам** (без «День 1», дат в названиях), чтобы по имени было понятно содержание; где есть логичный порядок — нумерация 01_, 02_ или «Модуль 1», «Модуль 2» в README.
- В начале каждого .txt — блок **Тема / Источник / Содержание**; тексты приведены к формату непрерывного чтения (маркеры слайдов и страниц удалены).
- Часть текстов оцифрована из сканов/презентаций; кодировка приведена к UTF-8. При артефактах OCR см. корневой README курса.
