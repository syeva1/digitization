# Оцифровка курсов Сколково

Материалы курсов Московской школы управления Сколково (BBA и др.), оцифрованные из презентаций и книг. Структура и названия приведены к тематическим (без привязки к дням); в каждой папке есть README с описанием. Курсы сделаны максимально автономными — по тексту и README можно разобраться без прохождения курса.

## Как пользоваться

- Зайдите в папку нужного курса и откройте **README.md** — там описание курса, для кого он, содержание и навигация по подпапкам.
- В каждой подпапке тоже есть **README.md** с темой блока и кратким описанием материалов.
- Названия папок и файлов — по **темам** (без «День 1», «Занятие 2», дат в названиях), чтобы по имени было понятно содержание.
- Часть текстов оцифрована из сканов/презентаций; при необходимости кодировка исправлена (UTF-8). Если в файле встречаются артефакты OCR или нечитаемые фрагменты — см. корневой README курса.

## Языки и программы
