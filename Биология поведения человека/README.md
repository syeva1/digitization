# Биология поведения человека

Курс (Behavioural Biology): биологические основы поведения.

## Для кого и зачем

Курс даёт базовые представления о биологии поведения: эволюция, нейробиология, поведенческие адаптации. Подходит для первого знакомства с темой без углублённой биологической подготовки.

## Содержание

Темы охватывают введение и шесть тематических блоков (по программе Behavioural Biology Syllabus 2023). Содержание каждого блока уточняется по материалам в папках Блок_1 … Блок_6.

## Как устроены материалы

- **Силлабус** — программа курса (Behavioural Biology Syllabus 2023).
- **Блок_1_Введение** — вводный блок.
- **Блок_2** … **Блок_6** — тематические блоки курса (порядок соответствует программе).

Рекомендуемый порядок: силлабус, затем Блок_1, далее Блок_2–6 по порядку.

## Язык и программа

Материалы на английском и/или русском. Курс входит в программу BBA.
