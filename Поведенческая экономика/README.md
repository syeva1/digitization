# Поведенческая экономика

Курс (BBA): экспериментальная и поведенческая экономика, эвристики, теория игр.

## Для кого и зачем

Курс знакомит с поведенческой и экспериментальной экономикой: введение (Usvitskiy), рациональность, эвристики (якорение, репрезентативность, доступность), искажения (endowment effect, confirmation bias, self-serving bias), теория игр, методы экспериментов, классические статьи (Chamberlin, Smith, Isaac-Walker, McCarter и др.). Подходит для интереса к экономике и психологии принятия решений.

## Содержание

Темы задаются материалами в корне: Introduction, Rationality, Behavioral Economics, Game Theory, Methods in Experimental Economics, эвристики и искажения (Anchoring, Representativeness, Availability, Endowment, Confirmation bias, Self-serving bias), литература (Chamberlin 1948, Smith 1962, Isaac Walker QJE 1988, McCarter et al 2020, behavioural_economics_text).

## Как устроены материалы

Все основные файлы курса расположены в корне папки «Поведенческая экономика». Имена файлов отражают темы (Introduction, Rationality, Game Theory, Anchoring_effect, и т.д.). Рекомендуемый порядок: Introduction → Behavioral Economics / Rationality → эвристики и искажения → Game Theory и Methods in Experimental Economics → статьи по экспериментам.

Материалы на английском. Курс: BBA.
