# Великие книги

Модуль: чтение и обсуждение классических текстов по менеджменту и лидерству.

## Для кого и зачем

Модуль знакомит с идеями из «великих книг»: от хорошего к великому (Коллинз), корпорация гениев, Генри Форд, дилемма инноватора, спиральная динамика, «никаких правил» и др. Подходит для осмысления лидерства и управления через классику.

## Содержание

Темы задаются книгами: Коллинз «От хорошего к великому», «Корпорация гениев», Форд «Моя жизнь», дилемма инноватора, спиральная динамика (Бек, Дон), материалы «никаких правил»; сопроводительные письма преподавателей (Прокопеня, Титарев, Коровкин).

## Как устроены материалы

- **Силлабус_Великие книги_2025-26** — программа модуля.
- **Великие книги_сопроводительное письмо_Прокопеня_2025-26**, **Сопроводительное письмо к модулю**, **Сопроводительное письмо_Владимир Коровкин_2025-26** — письма и введение в модуль.
- **Письмо_для_Великие_книги_СД_Дмитрий_Титарев** — материалы от Титарева.
- Папки по книгам: **dilemma-innovatora...**, **nikakih-pravil**, **spiralnaya-dinamika...**, **Ford_G._...**, **Kollinz_D._...**, **Kyetmell_Ye._...** — тексты или материалы по соответствующим книгам.

Рекомендуемый порядок: силлабус и сопроводительные письма, затем книги по программе.

## Язык и программа

Материалы на русском и английском. Модуль входит в программу BBA.
