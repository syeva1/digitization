# Психология

Курс Сколково: общая и когнитивная психология, эффект Мак-Гурка, итоговые проекты.

## Для кого и зачем

Курс даёт базовые понятия психологии: восприятие, когнитивные процессы, эффект Мак-Гурка (McGurk), релаксация ограничений и декомпозиция (Кноблих и др.). Подходит для первого знакомства с психологией в рамках BBA.

## Содержание

Темы: материалы в формате pdf+txt; итоговые проекты по первой части курса (в т.ч. проекты по эффекту Мак-Гурка); классические тексты (mcgurk1976, Constraint_Relaxation_and_Chunk_Decomposition).

## Как устроены материалы

- В корне: **mcgurk1976.txt**, **Constraint_Relaxation_and_Chunk_Decomposition_Кноблих и др.txt** — ключевые тексты.
- **Материалы_pdf_txt** — подборка материалов курса (pdf+txt).
- **Итоговые_проекты_часть_1** — итоговые проекты по первой части курса (в т.ч. блок по Мак-Гурку).

Рекомендуемый порядок: по программе курса; итоговые проекты — после прохождения первой части.

## Язык и программа

Материалы на русском и английском. Курс входит в программу BBA.
