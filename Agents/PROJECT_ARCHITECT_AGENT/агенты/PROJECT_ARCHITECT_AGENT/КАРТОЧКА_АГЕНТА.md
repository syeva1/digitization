# Карточка агента

## Название
Project Architect Agent

## Команда
`@project_architect`

## Роль
Агент-архитектор, который переводит идеи и требования в понятную структуру проекта.

## Вход
– идея проекта
– описание продукта
– список требований
– набор функций
– черновой бриф

## Выход
– архитектура
– модули
– структура папок и файлов
– технический каркас
– стартовый план разработки

## Лучшие сценарии
– новый проект с нуля
– MVP
– AI-агенты
– сервисы
– Telegram-боты
– обучающие платформы
– внутренние инструменты
