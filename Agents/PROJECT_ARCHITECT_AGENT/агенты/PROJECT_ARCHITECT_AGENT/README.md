# 🏗️ PROJECT ARCHITECT AGENT

Агент для проектирования структуры проектов, модулей и стартовой архитектуры.

---

# Что делает агент

Агент:

– анализирует идею проекта
– выделяет основные сущности и функции
– предлагает модули
– проектирует структуру папок и файлов
– помогает определить стек
– формирует стартовую документацию
– упрощает путь от идеи до нормального каркаса проекта

---

# Когда использовать

Используй агента, если нужно:

– придумать архитектуру нового проекта
– разложить идею на модули
– создать skeleton репозитория
– спроектировать структуру AI-агента
– оформить старт разработки приложения, сервиса, курса или бота

---

# Примеры использования

### Новый проект

```text
@project_architect
спроектируй структуру проекта для сервиса коротких AI-конспектов
```

### Разложение по модулям

```text
@project_architect
разбей эту идею на модули и предложи архитектуру
```

### Старт репозитория

```text
@project_architect
сделай skeleton проекта и структуру файлов
```

### Архитектура AI-агента

```text
@project_architect
спроектируй структуру Cursor-агента для восстановления курсов из PDF
```

---

# Что возвращает агент

На выходе агент может дать:

– краткое описание архитектуры
– список модулей
– структуру папок и файлов
– пояснение по каждому блоку
– план старта разработки
– README
– roadmap
– базовый каркас репозитория

---

# Логика работы

1. Анализирует цель проекта
2. Выделяет ключевые сущности
3. Определяет минимальную архитектуру
4. Проектирует структуру модулей
5. Строит файловую организацию
6. Даёт рекомендации по старту

---

# Результат

Ты получаешь не просто набор идей, а понятную основу проекта, с которой уже можно идти в разработку.
