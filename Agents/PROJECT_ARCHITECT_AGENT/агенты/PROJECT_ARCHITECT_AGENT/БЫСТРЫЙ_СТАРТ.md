# Быстрый старт

## 1. Куда положить правило

Скопируй файл:

`.cursor/rules/project_architect.mdc`

в проект Cursor.

## 2. Команда вызова

Файл называется `project_architect.mdc`, значит агент вызывается так:

```text
@project_architect
```

## 3. Проверка

Напиши в Cursor:

```text
@project_architect
привет
```

Если агент отвечает в своей роли – всё подключено правильно.

## 4. Первый полезный запрос

```text
@project_architect
спроектируй структуру проекта для платформы мини-курсов с AI-наставником
```
