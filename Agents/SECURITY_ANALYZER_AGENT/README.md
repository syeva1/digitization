# Security Analyzer Agent

Ищет уязвимости, проверяет безопасность кода.

## О агенте

**Security Analyzer Agent** — анализирует код на уязвимости и даёт рекомендации.

**Вызов:** `@security_analyzer` или `@безопасность`

**Вход:** код

**Выход:** уязвимости (тип, место, риск), рекомендации по исправлению
