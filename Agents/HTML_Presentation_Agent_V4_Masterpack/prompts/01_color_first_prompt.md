Сделай HTML-презентацию по этому материалу.
Перед генерацией выбери цветовой режим и коротко объясни, почему он подходит теме.
Потом выбери типографический пресет.
Не используй один шрифт на всё.
Сделай отдельный display-font, body-font и ui-font.
Если в проекте есть локальные шрифты – используй их через @font-face.
