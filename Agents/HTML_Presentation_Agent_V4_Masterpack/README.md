# HTML Presentation Agent – V4 Masterpack

Это расширенный masterpack на базе варианта 4.

## Что добавлено
– Montserrat как обязательный шрифт в библиотеке ролей
– подробный документ: какой шрифт зачем и для чего
– отдельная система свободы дизайна: что можно придумывать и добавлять
– жёсткие ограничения по safe area и вёрстке
– безопасный шаблон, чтобы ничего не вылезало за пределы слайда
– твой референсный HTML-кейс внутри архива
– примеры с разной типографикой

## Главное ограничение
В архив НЕ вложены бинарные font-файлы.
Вместо этого:
– добавлена архитектура
– добавлены роли шрифтов
– добавлены manifest и templates
– оставлена папка `assets/fonts/` для твоих `.woff2`

## Как использовать (в этом проекте)

Правило уже подключено: в корне проекта есть `.cursor/rules/presentation_agent.mdc`, он ссылается на эту папку.

**Активация в чате:** ввести `@presentation_agent` (или `@.cursor/rules/presentation_agent`) в поле сообщения и выбрать правило — тогда ответ пойдёт по инструкциям агента.

**Всегда применять:** в `.cursor/rules/presentation_agent.mdc` поставить `alwaysApply: true`, если хочешь, чтобы агент учитывался во всех ответах.

Ресурсы (docs, configs, templates, references, examples) лежат здесь; при желании заполняй `assets/fonts/` своими `.woff2`.
