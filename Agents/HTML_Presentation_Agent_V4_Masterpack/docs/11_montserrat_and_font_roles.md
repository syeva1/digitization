# 11. Montserrat and Font Roles

## Montserrat – обязательно в библиотеке
Montserrat – современный геометрический sans, который хорошо работает как:
– display-font для заголовков
– font для секционных маркеров
– font для крупных цифр и callouts
– secondary headline system в pitch deck

## Когда НЕ ставить Montserrat в body
Не делай Montserrat единственным шрифтом для больших массивов текста по умолчанию.
Для body чаще лучше:
– Inter
– IBM Plex Sans
– Karla
– Instrument Sans

## Примеры сочетаний

### Pairing 1 – Startup Pitch
display: Montserrat
body: Inter
ui: IBM Plex Mono

### Pairing 2 – Product Demo
display: Space Grotesk + Montserrat for numbers
body: IBM Plex Sans
ui: IBM Plex Mono

### Pairing 3 – Editorial / Strategy
display: Spectral or Lora
body: Inter
ui: IBM Plex Mono
Montserrat: only for section counters / highlight labels
