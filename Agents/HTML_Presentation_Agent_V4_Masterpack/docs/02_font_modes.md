# 02. Font Modes

## system-safe
Полностью автономный HTML без внешних подключений.
Плюсы:
– стабильно
– сразу работает
Минусы:
– ограниченное разнообразие

## local-fonts
Лучший режим.
В проект кладутся `.woff2`-файлы в `assets/fonts/`, а агент подключает их через `@font-face`.

## web-fonts
Использовать только если пользователь явно разрешил внешние зависимости.
