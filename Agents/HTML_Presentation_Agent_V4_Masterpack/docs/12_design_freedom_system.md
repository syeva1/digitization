# 12. Design Freedom System

## Главная идея
Агент должен чувствовать себя свободно, а не работать как генератор одного шаблона.

## Он может изобретать
– необычные композиции
– асимметричные сетки
– mix editorial + interface
– большие statement-экраны
– вертикальные sequences
– layered dashboard look
– cinematic reveal
– карточки разных размеров
– floating notes
– split blocks with unequal columns
– oversized counters
– stripes, dividers, side rails
– framed shells
– ticket-like labels
– command line inserts
– image placeholders
– texture overlays

## Но должен помнить
Свобода = вариативность, а не поломка композиции.
