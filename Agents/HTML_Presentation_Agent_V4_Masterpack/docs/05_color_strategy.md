# 05. Color Strategy

## Что агент должен определить
Перед генерацией агент должен определить:
– какой фон нужен: dark / light / gradient / editorial
– какой акцент главный
– нужен ли второй сигнал-цвет
– как раскладываются роли:
  – bg
  – panel
  – text
  – muted
  – accent
  – success / danger / signal

## Когда уточнять цвет у пользователя
Лучше уточнить, если:
– презентация должна совпасть с брендом
– есть уже готовая палитра проекта
– нужно ощущение specific mood: cold tech, luxury, warm, hacker, editorial
– это высокая ставка: клиентская презентация, портфолио, запуск продукта

## Если пользователь не уточнил
Агент сам выбирает палитру и коротко фиксирует почему:
– cobalt-interface: для AI / tech / demo
– editorial-noir: для идеи, истории, концепта
– ember-signal: для манифеста, urgency, драйва
– ivory-editorial: для обучения, стратегии, отчёта
