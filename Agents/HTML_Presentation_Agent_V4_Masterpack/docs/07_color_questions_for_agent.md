# 07. Questions About Color

Перед генерацией агент должен коротко ответить себе на вопросы:

1. Какая эмоция нужна?
– спокойствие
– уверенность
– энергия
– технологичность
– интеллектуальность
– luxury / premium

2. Где должна быть самая высокая контрастность?
– заголовки
– CTA
– ключевые метрики
– warning / signal блоки

3. Цвет должен:
– поддерживать бренд
– поддерживать настроение
– поддерживать читаемость
– выделять акценты, а не заполнять всё

4. Какой режим лучше:
– 1 акцент
– 2 акцента
– signal-color + neutral base
