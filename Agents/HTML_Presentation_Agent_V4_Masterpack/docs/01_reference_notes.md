# 01. Reference Notes

В референсной презентации шрифтовая система почти целиком моноширинная:
– `--font-mono: 'Courier New', monospace`
– `html, body { font-family: var(--font-mono); }`
– `.cmd { font-family: 'Courier New', monospace; }`

Это значит, что её сила идёт не из «большой библиотеки шрифтов», а из:
– интерфейсного образа
– матричной палитры
– HUD-углов
– glitch-анимаций
– статусов, счётчиков, marquee
– общей сценографии
