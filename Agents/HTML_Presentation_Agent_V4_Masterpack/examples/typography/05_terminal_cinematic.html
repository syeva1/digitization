Example 05 • Terminal Cinematic

Моно-подача тоже может быть сильной, если у неё есть атмосфера и контраст ролей

Этот пример ближе всех к референсному настроению – но уже с более аккуратным разделением ролей. Моно отвечает за характер и интерфейс, а основной текст остаётся удобным для чтения.

$ typography_mode --display mono --body readable --mood cinematic
