# Примеры HTML-презентаций с разной типографикой

Внутри 6 примеров:
– 01_cobalt_interface.html – жирный tech display + mono UI
– 02_editorial_noir.html – serif display + neutral sans body
– 03_ember_signal.html – condensed bold display
– 04_ivory_editorial.html – light editorial
– 05_terminal_cinematic.html – mono / terminal mood
– 06_soft_luxury_serif.html – мягкий luxury serif

Открой любой файл в браузере и сравни:
– тип display-font
– контраст между headline и body
– роль mono-подписей
– настроение, которое меняется только за счёт типографики и палитры
