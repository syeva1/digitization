# Агент продуктовых гипотез (Product Hypotheses)

Генерирует гипотезы улучшения продукта.

## О агенте

**Агент продуктовых гипотез** — генерирует гипотезы улучшения продукта для проверки.

**Вызов:** `@product_hypotheses` или `@продуктовые_гипотезы`

**Вход:** продукт, описание

**Выход:** гипотезы улучшения (что изменить, зачем, как проверить)

## Структура

```
Agents/PRODUCT_HYPOTHESES_AGENT/
├── _КОНТЕКСТ/
│   └── примеры/
├── _ЛОГИ/
└── README.md
```

## Использование

```
@product_hypotheses сгенерируй гипотезы для: [продукт]
```
