# Карточка агента

## Название
TEXT CONTENT ANALYST AGENT

## Команда вызова
`@text_content_analyst`

## Задача
Анализировать посты и тексты, находить сильные и слабые места, улучшать структуру, подачу и понятность.

## Основные функции
– анализ постов
– анализ статей
– анализ учебных текстов
– редакторский разбор
– сравнение версий текста
– предложения по улучшению
– перепаковка текста под цель

## Для кого подходит
– авторы постов
– редакторы
– преподаватели
– создатели курсов
– блогеры
– специалисты по контенту

## Результат
– конкретный разбор
– рекомендации
– улучшенная версия текста при необходимости
