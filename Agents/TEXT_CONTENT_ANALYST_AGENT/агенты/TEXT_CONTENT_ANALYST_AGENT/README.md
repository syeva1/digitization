# TEXT CONTENT ANALYST AGENT

Агент для анализа постов, статей, заметок, конспектов и любых других текстов.

## Что делает агент

Агент:
– анализирует смысл и цель текста
– оценивает структуру и читаемость
– показывает сильные и слабые стороны
– помогает улучшать подачу
– разбирает тексты под соцсети, обучение и контент
– может сравнивать несколько версий текста

## Когда использовать

Подходит для задач, где нужно:

– разобрать пост перед публикацией
– понять, почему текст слабый или сильный
– улучшить статью или заметку
– упаковать мысль понятнее
– проверить, насколько текст цепляет
– адаптировать материал под обучение или контент

## Примеры использования

### Разобрать пост

```text
@text_content_analyst
разбери этот пост и скажи, что в нём работает, а что нет
```

### Проанализировать учебный текст

```text
@text_content_analyst
оцени, насколько этот материал понятен для ученика
```

### Улучшить текст

```text
@text_content_analyst
разбери этот текст и перепиши его сильнее, сохранив смысл
```

### Сравнить два варианта

```text
@text_content_analyst
сравни эти два текста и скажи, какой лучше для Telegram-поста
```

## Что получает пользователь

На выходе:

– понятный разбор
– сильные и слабые стороны
– рекомендации по улучшению
– при необходимости улучшенную версию текста
– более ясное понимание качества контента
