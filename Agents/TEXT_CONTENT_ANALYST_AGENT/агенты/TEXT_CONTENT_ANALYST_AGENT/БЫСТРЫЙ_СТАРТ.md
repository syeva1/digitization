# Быстрый старт

## 1. Куда положить правило

Создай файл:

`.cursor/rules/text_content_analyst.mdc`

## 2. Как вызвать агента

Используй команду:

`@text_content_analyst`

## 3. Примеры запросов

### Анализ поста

```text
@text_content_analyst
разбери этот пост для Telegram и скажи, где он проседает
```

### Анализ статьи

```text
@text_content_analyst
проанализируй эту статью по структуре, логике и полезности
```

### Анализ конспекта

```text
@text_content_analyst
посмотри, насколько этот текст похож на хороший учебный материал
```

### Улучшение текста

```text
@text_content_analyst
разбери и перепиши этот текст так, чтобы он стал яснее и сильнее
```
