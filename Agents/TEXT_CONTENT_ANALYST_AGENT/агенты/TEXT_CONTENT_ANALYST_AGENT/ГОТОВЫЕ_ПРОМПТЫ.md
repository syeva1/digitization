# Готовые промпты

## 1. Анализ поста

```text
@text_content_analyst
разбери этот пост по структуре, силе начала, эмоциональности и желанию дочитать до конца
```

## 2. Редакторский разбор

```text
@text_content_analyst
сделай редакторский разбор этого текста: что в нём работает, что мешает, что нужно усилить
```

## 3. Анализ учебного материала

```text
@text_content_analyst
оцени этот текст как материал для ученика: насколько он понятен, логичен и удобен для изучения
```

## 4. Улучшение текста

```text
@text_content_analyst
разбери этот текст и перепиши его сильнее, но без потери смысла
```

## 5. Сравнение двух вариантов

```text
@text_content_analyst
сравни эти два варианта поста и скажи, какой сильнее и почему
```

## 6. Адаптация под соцсети

```text
@text_content_analyst
посмотри на этот текст и скажи, как сделать его более цепляющим для Telegram
```
