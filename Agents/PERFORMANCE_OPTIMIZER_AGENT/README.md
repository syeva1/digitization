# Performance Optimizer Agent

Анализирует производительность, предлагает оптимизации.

## О агенте

**Performance Optimizer Agent** — оптимизирует код и алгоритмы: находит узкие места, предлагает улучшения.

**Вызов:** `@performance_optimizer` или `@оптимизация`

**Вход:** код, алгоритм

**Выход:** узкие места, предложения по оптимизации, улучшенный код
