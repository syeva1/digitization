# Агенты

Папка с агентами (правилами и наборами инструкций) для Cursor в этом проекте. Все агенты перед работой задают уточняющие вопросы: зачем это нужно и для чего будет использоваться результат.

## Подпапки

- **HTML_Presentation_Agent_V4_Masterpack/** — агент для HTML-презентаций: типографика, палитры, safe area, шаблоны, примеры
- **ДЕМО_АГЕНТ_УБОРЩИК/** — уборщик документов: организация файлов и папок, поиск дубликатов, система самообучения (кейсы в _КОНТЕКСТ/примеры, уроки в _ЛОГИ)
- **TEXT_CONTENT_ANALYST_AGENT/** — анализ постов, статей, заметок: разбор по смыслу, структуре, подаче, улучшения
- **PROMPT_GENERATOR_AGENT/** — создание и улучшение промптов для Cursor, ChatGPT, Claude, .mdc-правила
- **PROJECT_ARCHITECT_AGENT/** — архитектура проектов: структура папок, модули, создаёт папки и базовые файлы, README, roadmap
- **COURSE_RECOVERY_AGENT/** — восстановление курсов после PDF/OCR, битая кодировка, структурирование материалов
- **GENSPARK_PROMPT_AGENT/** — промпты для Genspark: один промпт на слайд, дизайн-спецификация, цвета, типографика, композиция
- **DISTRIBUTION_AGENT/** — распределение файлов из 07_Distribution по структуре проекта, переименование по стандарту, обновление README
- **REPORT_ANALYST_AGENT/** — анализ отчётов Олеси и учеников: риски, статусы CRM, сводка для Антона
- **PRODUCT_PACKAGING_AGENT/** — упаковка продукта: идея + описание + функции → позиционирование, УТП, ценностное предложение, структура лендинга
- **COMPETITOR_ANALYST_AGENT/** — анализ конкурентов: сайт + описание продукта → сильные/слабые стороны, отличия, идеи улучшения
- **BUSINESS_MODEL_AGENT/** — бизнес-модели: идея → бизнес-модель, источники дохода, unit-экономика, гипотезы монетизации
- **COURSE_CREATOR_AGENT/** — создание курса: тема → структура, модули, уроки, задания, проекты
- **DOCUMENT_SUMMARIZER_AGENT/** — резюмирование: длинный текст/книга/лекция → резюме, ключевые идеи, тезисы
- **KNOWLEDGE_STRUCTURER_AGENT/** — структурирование знаний: хаотичные заметки → структура, карта знаний, категории, связки
- **POST_CREATOR_AGENT/** — посты: мысль/тезисы → пост, структура, крючок
- **TEXT_EDITOR_AGENT/** — редактура текста: улучшает текст, убирает мусор, делает стиль лучше
- **ADS_CREATOR_AGENT/** — реклама: идеи рекламы, тексты объявлений, креативы
- **LEAD_MAGNET_AGENT/** — лидмагниты: идеи бесплатных продуктов, чек-листы, мини-гайды
- **STARTUP_IDEAS_AGENT/** — идеи стартапов: тренды/технологии → идеи продуктов
- **IDEA_IMPROVER_AGENT/** — улучшение идей: усиливает идею, находит слабые места
- **DECISION_MAKER_AGENT/** — принятие решений: анализирует варианты, плюсы/минусы, предлагает выбор
- **QUESTION_GENERATOR_AGENT/** — генерация вопросов: задаёт вопросы по теме, помогает глубже думать
- **STRATEGIC_THINKING_AGENT/** — стратегическое мышление: анализирует ситуацию, предлагает стратегии
- **MVP_AGENT/** — MVP: идея → минимальный продукт, must-have функции, приоритизация, критерий готовности
- **MARKET_SEARCH_AGENT/** — поиск рынков: продукт + ЦА → новые сегменты, ниши, потенциал, барьеры входа
- **PRODUCT_HYPOTHESES_AGENT/** — продуктовые гипотезы: продукт → гипотезы улучшения, как проверить, приоритеты
- **CODE_REFACTOR_AGENT/** — рефакторинг кода: убирает дублирование, улучшает читаемость, разбивает функции
- **BUG_HUNTER_AGENT/** — поиск багов: код → потенциальные баги, место, причина, рекомендация
- **DB_ARCHITECT_AGENT/** — архитектор БД: домен → таблицы, связи, индексы, SQL или диаграмма
- **SECURITY_ANALYZER_AGENT/** — анализ безопасности: код → уязвимости, риск, как исправить
- **PERFORMANCE_OPTIMIZER_AGENT/** — оптимизация производительности: код, алгоритм → узкие места, оптимизации, улучшенный код

## Как активировать агента

Все правила доступны из корня проекта: в корне репозитория создана папка **`.cursor/rules/`** со симлинками на каждый агент. Cursor подхватывает правила из корня, поэтому все агенты работают при вызове через **`@`** в чате.

**Как вызвать:** в чате наберите `@` и начните вводить имя правила (например `@text_content_analyst`, `@prompt_generator`, `@course_recovery`, `@уборщик`). Выберите нужный агент из списка.

### Презентации (presentation_agent)

Правило: `.cursor/rules/presentation_agent.mdc`.

**Включить в чате:** `@` → `presentation` или `presentation_agent`. Ответ идёт по инструкциям агента (слайды, типографика, палитры, вёрстка).

**Авто-подключение:** при открытии или создании HTML-файлов в `03_Контекст` или `04_Промпты` правило может предлагаться автоматически (globs).

Ресурсы: `Agents/HTML_Presentation_Agent_V4_Masterpack/`.

### Уборщик документов (уборщик)

Правило: `.cursor/rules/уборщик.mdc`.

**Включить в чате:** `@уборщик` и задачу (например: «наведи порядок в папке 07_Distribution» или «создай структуру для проекта X»). Агент предложит план, после подтверждения выполнит и попросит обратную связь; успешные кейсы — в `Agents/ДЕМО_АГЕНТ_УБОРЩИК/_КОНТЕКСТ/примеры/`, уроки — в `_ЛОГИ/`.

### Аналитик текста (text_content_analyst)

Правило: `.cursor/rules/text_content_analyst.mdc`.

**Включить в чате:** `@text_content_analyst`. Разбор постов, статей, заметок: главная мысль, сильные/слабые места, улучшения, оценка. Кейсы — в `Agents/TEXT_CONTENT_ANALYST_AGENT/_КОНТЕКСТ/примеры/`, уроки — в `_ЛОГИ/`.
