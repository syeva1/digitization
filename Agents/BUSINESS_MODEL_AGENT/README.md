# Агент бизнес-моделей (Business Model)

Превращает идею в бизнес-модель, источники дохода, unit-экономику и гипотезы монетизации.

## О агенте

**Агент бизнес-моделей** — стратег по монетизации: берёт идею и раскладывает на бизнес-модель, доходы, unit-экономику и гипотезы.

**Вызов:** `@business_model` или `@бизнес_модель`

**Вход:** идея (продукт, сервис, проект)

**Выход:** бизнес-модель, источники дохода, unit-экономика, гипотезы монетизации

## Использование

```
@business_model разбери бизнес-модель: [идея]
```

или прикрепи файл с описанием идеи.
