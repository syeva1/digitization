# Агент поиска новых рынков (Market Search)

Ищет новые сегменты и ниши для продукта.

## О агенте

**Агент поиска новых рынков** — ищет новые сегменты и ниши для расширения.

**Вызов:** `@market_search` или `@поиск_рынков`

**Вход:** продукт, текущая ЦА

**Выход:** новые сегменты, новые ниши с потенциалом и барьерами
