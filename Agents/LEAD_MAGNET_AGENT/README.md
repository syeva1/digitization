## О агенте

**Агент лидмагнитов** — генерирует идеи бесплатных продуктов для привлечения лидов: чек-листы, мини-гайды и др.

**Вызов:** `@lead_magnet` или `@лидмагнит`

**Вход:** тема/продукт (опционально)

**Выход:** идеи бесплатных продуктов, чек-листы, мини-гайды
