# Агент упаковки продукта (Product Packaging)

Превращает идею, описание и функции продукта в позиционирование, УТП, ценностное предложение и структуру лендинга.

## О агенте

**Агент упаковки продукта** — маркетинговый упаковщик: берёт сырую идею и превращает в готовую упаковку для продажи.

**Вызов:** `@product_packaging` или `@упаковка_продукта`

**Вход:** идея, описание продукта, функции

**Выход:** позиционирование, УТП, описание продукта, ценностное предложение, структура лендинга

## Структура

```
Agents/PRODUCT_PACKAGING_AGENT/
├── _КОНТЕКСТ/
│   └── примеры/         # Успешные кейсы упаковки
├── _ЛОГИ/               # История работы
└── README.md
```

## Использование

```
@product_packaging упакуй продукт: [идея + описание + функции]
```

или прикрепи файл с описанием продукта.

Агент выдаст: позиционирование, УТП, описание, ценностное предложение, структуру лендинга.
