# Агент идей стартапов (Startup Ideas)

Генерирует идеи продуктов из трендов и технологий.

## О агенте

**Агент идей стартапов** — берёт тренды и технологии и генерирует идеи продуктов.

**Вызов:** `@startup_ideas` или `@идеи_стартапов`

**Вход:** тренды, технологии

**Выход:** идеи продуктов

## Структура
