# Агент MVP (MVP)

Определяет минимальный продукт и ключевые функции.

## О агенте

**Агент MVP** — помогает определить минимальный продукт и ключевые функции для запуска.

**Вызов:** `@mvp` или `@мвп`

**Вход:** идея продукта, описание

**Выход:** минимальный продукт, ключевые функции, приоритизация, критерий готовности
