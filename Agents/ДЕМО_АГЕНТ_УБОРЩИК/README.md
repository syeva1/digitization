## 🚀 УСТАНОВКА (5 минут)

### Шаг 1: Создай папку для агента

```bash
mkdir -p агенты/УБОРЩИК_ДОКУМЕНТОВ/_КОНТЕКСТ/примеры
mkdir -p агенты/УБОРЩИК_ДОКУМЕНТОВ/_ЛОГИ
```

### Шаг 2: Скопируй файл правил

Скопируй файл `уборщик_документов.mdc` в папку `.cursor/rules/`:

```bash
cp уборщик_документов.mdc .cursor/rules/уборщик.mdc
```

**ВАЖНО:** Имя файла = команда для вызова. Если файл называется `уборщик.mdc`, вызов через `@уборщик`.

### Шаг 3: Перезапусти Cursor

1. Закрой Cursor
2. Открой снова
3. Агент готов к работе!

### Шаг 4: Протестируй агента

В Cursor набери:
```
@уборщик привет
```

Агент должен ответить и предложить помощь.

## 🎯 СИСТЕМА САМООБУЧЕНИЯ

### Как это работает:

**1. Ты даёшь задачу**
```
@уборщик организуй папку Projects
```

**2. Агент выполняет**
- Анализирует
- Предлагает решение
- Выполняет после подтверждения

**3. Агент запрашивает обратную связь**
```
Задача выполнена. Оцени результат:
- Отлично (использую как пример)
- Хорошо (работает)
- Нужны правки (что исправить?)
- Плохо (что не так?)
```

**4. Агент сохраняет знания**

Если "Отлично" или "Хорошо":
- Кейс сохраняется в `_КОНТЕКСТ/примеры/`
- Используется в будущих задачах

Если "Нужны правки":
- Урок записывается в `_ЛОГИ/`
- Агент учитывает при следующей задаче

**5. Агент становится лучше**

День 1: Работает хорошо  
День 2: Работает лучше (использует опыт)  
День 3: Работает отлично (накоплен опыт)

## 🛠️ РЕШЕНИЕ ПРОБЛЕМ

### Проблема: Агент не отвечает на `@уборщик`

**Решение:**
1. Проверь что файл называется `уборщик.mdc` (без символов)
2. Проверь что файл в папке `.cursor/rules/`
3. Перезапусти Cursor

### Проблема: Агент не сохраняет логи

**Решение:**
1. Проверь что папка `агенты/УБОРЩИК_ДОКУМЕНТОВ/_ЛОГИ/` существует
2. Дай агенту разрешение создавать файлы
3. Проверь что ты даёшь обратную связь после задачи

### Проблема: Агент не использует опыт

**Решение:**
1. Убедись что файл правил включает систему самообучения
2. Проверь что кейсы сохраняются в `_КОНТЕКСТ/примеры/`
3. Дай агенту несколько задач для накопления опыта
