# Агент редактуры текста (Text Editor)

Улучшает текст: убирает мусор, делает стиль лучше.

## О агенте

**Агент редактуры текста** — редактор: улучшает текст, убирает воду и канцелярит, улучшает читаемость и стиль.

**Вызов:** `@text_editor` или `@редактура`

**Вход:** текст

**Выход:** улучшенный текст (без мусора, лучший стиль)
