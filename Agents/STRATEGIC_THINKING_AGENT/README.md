## О агенте

**Агент стратегического мышления** — анализирует ситуацию и предлагает стратегии с плюсами/минусами.

**Вызов:** `@strategic_thinking` или `@стратегия`

**Вход:** ситуация

**Выход:** анализ, стратегии
