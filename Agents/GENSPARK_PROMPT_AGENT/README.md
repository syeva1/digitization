# Genspark Prompt Agent

Агент для создания промптов для Genspark — инструмента для гибкого создания презентаций.

## Назначение

Превращает сырой материал (тексты, заметки, документы) в детальные промпты для Genspark — по одному промпту на слайд, с полной дизайн-спецификацией: блоки, шрифты, цвета, композиция, визуальные элементы.

**Главный приоритет:** дизайн. Максимальная свобода творчества, разнообразие визуальных решений, никакой шаблонности.

## Активация

Правило: `.cursor/rules/genspark_prompt.mdc`

**Включить в чате:** `@genspark_prompt`

## Подпапки

- **_КОНТЕКСТ/** — успешные кейсы промптов для Genspark
- **_ЛОГИ/** — уроки и обратная связь

## Особенности

- Перед работой всегда задаёт вопросы: цель, аудитория, цвета, стиль, количество слайдов
- Выдаёт промпты по одному на слайд — готовые для копирования в Genspark
- Указывает для каждого слайда: композицию, типографику, цвета, блоки, визуальные элементы
- Широкий набор приёмов: асимметрия, split-screen, timeline, big number, quote slides, editorial layouts и др.
