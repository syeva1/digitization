# Агент генерации вопросов (Question Generator)

Задаёт вопросы по теме, помогает глубже думать.

## О агенте

**Агент генерации вопросов** — задаёт вопросы по теме, помогает углубить мышление и рефлексию.

**Вызов:** `@question_generator` или `@вопросы`

**Вход:** тема

**Выход:** вопросы для углубления мышления
