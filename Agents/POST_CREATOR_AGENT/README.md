# Агент постов (Post Creator)

Превращает мысль и тезисы в пост со структурой и крючком.

## О агенте

**Агент постов** — берёт мысль или тезисы и создаёт пост с крючком и структурой.

**Вызов:** `@post_creator` или `@пост`

**Вход:** мысль, тезисы

**Выход:** пост, структура, крючок

## Структура

├── _КОНТЕКСТ/

## Использование

```
@post_creator сделай пост из: [мысль или тезисы]
```
