# Агент создания курса (Course Creator)

Превращает тему в структуру курса: модули, уроки, задания, проекты.

## О агенте

**Агент создания курса** — методист: берёт тему и раскладывает на структуру курса с модулями, уроками, заданиями и проектами.

**Вызов:** `@course_creator` или `@создание_курса`

**Вход:** тема курса

**Выход:** структура курса, модули, уроки, задания, проекты

## Структура

```
Agents/COURSE_CREATOR_AGENT/
├── _КОНТЕКСТ/
│   └── примеры/         # Успешные кейсы
├── _ЛОГИ/               # История работы
└── README.md
```

## Использование

```
@course_creator создай курс по теме: [тема]
```
