# Database Architect Agent

Проектирует БД: таблицы, связи, индексы.

## О агенте

**Database Architect Agent** — проектирует структуру базы данных для домена.

**Вызов:** `@db_architect` или `@архитектор_бд`

**Вход:** описание домена (например, интернет-магазин)

**Выход:** таблицы, поля, связи (FK), индексы, SQL или диаграмма
