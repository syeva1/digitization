# Быстрый старт

## 1. Куда положить файл правил

Создай файл:

```text
.cursor/rules/course_recovery.mdc
```

и вставь в него содержимое из архива.

## 2. Куда положить папку агента

Скопируй папку:

```text
агенты/COURSE_RECOVERY_AGENT
```

в свой проект.

## 3. Перезапусти Cursor

После добавления файлов перезапусти Cursor, чтобы правила подхватились корректно.

## 4. Проверь запуск

Вызови агента так:

```text
@course_recovery привет
```

Если агент подхватился, можешь давать ему реальные задачи.

## 5. Стартовые задачи

```text
@course_recovery
почисти этот текст после кривого PDF и сделай из него понятный урок
```

```text
@course_recovery
восстанови структуру курса и предложи правильные названия файлов
```

```text
@course_recovery
собери из этих фрагментов нормальный модуль курса
```
