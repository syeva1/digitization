# Карточка агента

## Название
COURSE RECOVERY AGENT

## Команда
`@course_recovery`

## Назначение
Восстанавливает учебные материалы после плохого извлечения из PDF, OCR, битой кодировки и хаотичной структуры файлов.

## Главная ценность
Превращает сломанные материалы в понятный, чистый, логично организованный курс.

## Основные режимы
1. Cleanup
2. Structure
3. Course Packaging
4. File Organization

## Что получает пользователь
– чистый текст
– восстановленную структуру
– нормальные заголовки
– понятные названия файлов
– аккуратную упаковку курса
