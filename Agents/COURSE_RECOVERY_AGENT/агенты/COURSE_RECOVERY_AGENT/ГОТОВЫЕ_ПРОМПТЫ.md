# Готовые промпты

## Очистка текста

```text
@course_recovery
почисти этот текст после PDF extraction: исправь кракозябры, склей абзацы и верни нормальную читаемую версию
```

## Восстановление урока

```text
@course_recovery
преврати этот сломанный текст в полноценный урок: выдели заголовок, блоки, примеры и итоги
```

## Упаковка модуля

```text
@course_recovery
из этих фрагментов собери понятную структуру модуля курса
```

## Переименование файлов

```text
@course_recovery
предложи нормальные названия для этих файлов и собери логичную структуру папок
```

## Полное восстановление

```text
@course_recovery
сначала восстанови текст, потом предложи структуру курса, затем дай понятные названия файлов
```
