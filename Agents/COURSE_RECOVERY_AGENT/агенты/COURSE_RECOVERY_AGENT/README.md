# COURSE RECOVERY AGENT

Агент для восстановления учебных материалов после плохого извлечения из PDF, OCR и битой кодировки.

## Что делает агент

Агент:
– исправляет кракозябры и битую кодировку
– убирает мусор после OCR и PDF extraction
– склеивает разорванные строки и абзацы
– восстанавливает заголовки, списки и структуру урока
– превращает хаотичный текст в понятный учебный материал
– помогает переименовать файлы и навести порядок в папках курса

## Когда использовать

Подходит, если:
– текст был плохо извлечён из PDF
– документ криво распознался через OCR
– в файлах есть мусорные символы и сломанные буквы
– курс распался на непонятные куски
– названия файлов и папок выглядят хаотично

## Команда вызова

```text
@course_recovery
```

## Примеры использования

### 1. Очистка текста

```text
@course_recovery
почисти этот текст после PDF и восстанови нормальную структуру урока
```

### 2. Восстановление урока

```text
@course_recovery
преврати этот сломанный текст в нормальный урок
```

### 3. Переименование файлов

```text
@course_recovery
предложи нормальные названия файлов и структуру папок для этого курса
```

### 4. Полная упаковка курса

```text
@course_recovery
из этих файлов и фрагментов собери аккуратную структуру курса
```

## Как работает

1. Анализирует, что именно сломано
2. Очищает текст и кодировку
3. Восстанавливает читаемую структуру
4. Выделяет логику уроков и модулей
5. Предлагает понятные имена и итоговую упаковку

## Результат

На выходе можно получить:
– очищенный текст
– восстановленный урок
– структуру модуля
– схему переименования файлов
– предложенную структуру папок курса
