# Агент резюмирования документов (Document Summarizer)

Резюмирует длинные тексты, книги, лекции: краткое резюме, ключевые идеи, тезисы.

## О агенте

**Агент резюмирования документов** — берёт длинный текст и выдаёт сжатое резюме, ключевые идеи и тезисы.

**Вызов:** `@document_summarizer` или `@резюме`

**Вход:** длинный текст, книга, лекция

**Выход:** краткое резюме, ключевые идеи, тезисы

## Структура

```
Agents/DOCUMENT_SUMMARIZER_AGENT/
├── _КОНТЕКСТ/
│   └── примеры/
├── _ЛОГИ/
└── README.md
```

## Использование

```
@document_summarizer резюмируй: [прикрепи текст или файл]
```
