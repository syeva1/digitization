# Агент улучшения идей (Idea Improver)

Усиливает идею и находит слабые места.

## О агенте

**Агент улучшения идей** — берёт идею и усиливает её, находит слабые места, даёт рекомендации.

**Вызов:** `@idea_improver` или `@улучшение_идеи`

**Вход:** идея

**Выход:** усиленная идея, слабые места
