# Агент рекламы (Ads Creator)

Создаёт идеи рекламы, тексты объявлений, креативы.

## О агенте

**Агент рекламы** — креативщик: генерирует идеи рекламы, тексты объявлений и описания креативов.

**Вызов:** `@ads_creator` или `@реклама`

**Вход:** продукт/услуга (опционально)

**Выход:** идеи рекламы, тексты объявлений, креативы

## Структура

