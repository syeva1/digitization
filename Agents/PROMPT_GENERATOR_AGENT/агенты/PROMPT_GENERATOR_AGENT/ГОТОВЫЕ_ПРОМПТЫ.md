# ГОТОВЫЕ ПРОМПТЫ

## 1. Промпт для улучшения промпта

```text
@prompt_generator
улучши этот промпт: сделай его конкретнее, убери двусмысленность, добавь ограничения и формат результата
```

## 2. Промпт для системной роли

```text
@prompt_generator
создай системный промпт для агента, который анализирует и структурирует учебные тексты
```

## 3. Промпт для Cursor

```text
@prompt_generator
сделай .mdc-правило для агента, который восстанавливает курс из криво извлечённого PDF-текста
```

## 4. Промпт для ChatGPT

```text
@prompt_generator
создай промпт для ChatGPT, который превращает заметки в понятную статью с подзаголовками и примерами
```

## 5. Промпт для упаковки агента

```text
@prompt_generator
упакуй агента для html-страниц: дай название, системный промпт, .mdc, структуру папок и примеры запросов
```
