# БЫСТРЫЙ СТАРТ

## Установка

1. Скопируй файл:

`.cursor/rules/prompt_generator.mdc`

2. Создай папку агента:

`агенты/PROMPT_GENERATOR_AGENT/`

3. Перезапусти Cursor

4. Проверь вызов:

```text
@prompt_generator привет
```

## Базовые сценарии

### Создать промпт

```text
@prompt_generator
сделай промпт для ChatGPT, который превращает заметки в статью
```

### Улучшить промпт

```text
@prompt_generator
перепиши этот промпт так, чтобы ответ был точнее и структурнее
```

### Упаковать агента

```text
@prompt_generator
собери AI-агента для генерации html-страниц
```
