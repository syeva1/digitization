# КАРТОЧКА АГЕНТА

## Название
PROMPT GENERATOR AGENT

## Назначение
Создание, усиление и упаковка промптов под разные платформы и задачи.

## Основные функции

- prompt engineering
- улучшение пользовательских запросов
- адаптация под Cursor / ChatGPT / Claude
- генерация системных инструкций
- сборка `.mdc`-правил
- упаковка AI-агентов

## Формат результата

Агент может возвращать:

- готовый промпт
- несколько версий промпта
- системный промпт
- `.mdc`
- структуру агента
- примеры использования

## Ценность

Агент помогает быстро превращать сырые идеи в рабочие инструкции, которые дают более качественный и предсказуемый результат.
