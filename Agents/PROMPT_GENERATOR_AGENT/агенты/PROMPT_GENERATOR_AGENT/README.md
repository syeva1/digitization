# PROMPT GENERATOR AGENT

Агент для создания, улучшения и упаковки промптов под Cursor, ChatGPT, Claude и AI-агентов.

## Что делает агент

Агент:

- создаёт промпты с нуля
- улучшает слабые промпты
- адаптирует промпты под разные платформы
- помогает собирать системные инструкции
- делает `.mdc`-правила для Cursor
- упаковывает идеи в формат готового AI-агента

## Для чего подходит

- промпты для Cursor
- системные промпты для агентов
- user-prompts для ChatGPT
- правила для `.mdc`
- библиотека шаблонов под типовые задачи
- улучшение существующих инструкций

## Команда вызова

Файл правил:

`.cursor/rules/prompt_generator.mdc`

Вызов в Cursor:

`@prompt_generator`

## Примеры использования

### 1. Создать промпт с нуля

```text
@prompt_generator
сделай сильный промпт для агента, который превращает сырой текст в конспект
```

### 2. Улучшить слабый промпт

```text
@prompt_generator
улучши этот промпт, сделай его более точным и стабильным
```

### 3. Сделать `.mdc`

```text
@prompt_generator
преврати эту идею в готовое .mdc-правило для Cursor
```

### 4. Упаковать агента

```text
@prompt_generator
собери агента для анализа текстов: название, системный промпт, .mdc, примеры запросов
```

## Логика работы

1. Анализирует цель
2. Определяет платформу
3. Выделяет роль, контекст, ограничения
4. Строит сильную структуру промпта
5. Возвращает готовую версию для копирования

## Результат

На выходе пользователь получает:

- готовый промпт
- улучшенную структуру инструкций
- более стабильный результат модели
- шаблон для повторного использования
