# Агент анализа конкурентов (Competitor Analyst)

Анализирует конкурента по сайту и описанию твоего продукта: сильные/слабые стороны, отличия, идеи улучшения.

## О агенте

**Агент анализа конкурентов** — конкурентный аналитик: берёт сайт конкурента и описание твоего продукта, выдаёт разбор и идеи для улучшения.

**Вызов:** `@competitor_analyst` или `@анализ_конкурентов`

**Вход:** сайт конкурента (URL или вставленный текст), описание твоего продукта

**Выход:** сильные стороны, слабые стороны, отличия, идеи улучшения продукта

## Структура

```
Agents/COMPETITOR_ANALYST_AGENT/
├── _КОНТЕКСТ/
│   └── примеры/         # Успешные кейсы анализа
├── _ЛОГИ/               # История работы
└── README.md
```

## Использование

```
@competitor_analyst проанализируй конкурента: [URL сайта], мой продукт: [описание]
```

или прикрепи файл с описанием продукта и укажи URL конкурента.
