# Агент структурирования знаний (Knowledge Structurer)

Превращает хаотичные заметки в структуру, карту знаний, категории и связки.

## О агенте

**Агент структурирования знаний** — берёт разрозненные заметки и создаёт структуру, карту знаний, категории и связи между концептами.

**Вызов:** `@knowledge_structurer` или `@структурирование_знаний`

**Вход:** хаотичные заметки

**Выход:** структура, карта знаний, категории, связки

## Структура

```
Agents/KNOWLEDGE_STRUCTURER_AGENT/
├── _КОНТЕКСТ/
│   └── примеры/
├── _ЛОГИ/
└── README.md
```

## Использование

```
@knowledge_structurer структурируй заметки: [прикрепи или вставь]
```
