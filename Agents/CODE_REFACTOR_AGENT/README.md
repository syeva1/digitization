# Code Refactor Agent

Улучшает код: убирает дублирование, делает читаемым, делит большие функции.

## О агенте

**Code Refactor Agent** — рефакторит код: убирает дублирование, улучшает читаемость, разбивает большие функции.

**Вызов:** `@code_refactor` или `@рефакторинг`

**Вход:** код

**Выход:** улучшенный код, упрощённая структура

## Структура

```
Agents/CODE_REFACTOR_AGENT/
├── _КОНТЕКСТ/
│   └── примеры/
├── _ЛОГИ/
└── README.md
```

## Использование

```
@code_refactor улучши этот код и упрости структуру
```
