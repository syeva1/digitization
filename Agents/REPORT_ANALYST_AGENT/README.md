# Агент анализа отчётов (Report Analyst)

Анализ отчётов Олеси и недельных отчётов учеников: сводка, риски, предложения по статусам CRM.

## О агенте

**Агент анализа отчётов** — разбирает операционные отчёты и отчёты учеников, выделяет риски, предлагает статусы (Зелёный/Жёлтый/Красный) и формирует сводку для Антона.

**Вызов:** `@report_analyst` или `@анализ_отчётов`

**ROI:** Экономия времени на ручном разборе, единый формат сводки, не пропустить риски

## Использование

```
@report_analyst проанализируй отчёт за 25 февраля
```

или

```
@анализ_отчётов [прикрепи файл отчёта]
```

Агент:
1. Прочитает отчёт (операционный или ученический)
2. Выделит упоминания учеников, риски, проблемы
3. Предложит статусы и обновления для CRM
4. Сформирует краткую сводку для Антона
