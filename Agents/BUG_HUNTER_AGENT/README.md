# Bug Hunter Agent

Анализирует код, находит потенциальные баги, объясняет причины.

## О агенте

**Bug Hunter Agent** — ищет потенциальные ошибки в коде и объясняет причины.

**Вызов:** `@bug_hunter` или `@поиск_багов`

**Вход:** код

**Выход:** потенциальные баги с местом, причиной, рекомендацией
