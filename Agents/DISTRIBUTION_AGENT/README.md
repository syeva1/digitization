# Агент распределения (Distribution)

Автоматическое распределение файлов из `07_Distribution` по структуре проекта.

## О агенте

**Агент распределения** — распределяет загруженные файлы по нужным папкам, переименовывает по стандарту проекта и обновляет README с историей.

**Вызов:** `@distribution` или `@распределение`

**ROI:** Экономия времени на ручной раскладке файлов, единый стандарт нейминга

## Структура

```
Agents/DISTRIBUTION_AGENT/
├── _КОНТЕКСТ/
│   └── примеры/         # Успешные кейсы распределения
├── _ЛОГИ/               # История работы
└── README.md
```

## Использование

```
@distribution распредели файлы из 07_Distribution
```

или

```
@распределение
```

Агент:
1. Проверит содержимое `07_Distribution`
2. Предложит план распределения с переименованием
3. Выполнит после подтверждения
4. Обновит `07_Distribution/README.md` с историей
