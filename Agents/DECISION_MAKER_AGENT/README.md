# Агент принятия решений (Decision Maker)

Анализирует варианты, делает плюсы/минусы, предлагает выбор.

## О агенте

**Агент принятия решений** — анализирует варианты, делает плюсы/минусы, предлагает выбор с обоснованием.

**Вызов:** `@decision_maker` или `@решение`

**Вход:** ситуация, варианты

**Выход:** анализ вариантов, плюсы/минусы, рекомендация
