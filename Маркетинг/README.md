# Маркетинг

Курс: основы маркетинга (2024).

## Для кого и зачем

Курс даёт базовые концепции маркетинга: рынки, потребители, позиционирование, коммуникация. Подходит для первого знакомства с маркетингом в рамках BBA.

## Содержание и материалы

Содержание задаётся силлабусом (Силлабус_Маркетинг 2024). Структура папок и файлов уточняется по списку в директории курса.

## Как устроены материалы

Материалы курса размещены в папке курса; при наличии подпапок — каждая с собственным README. Рекомендуемый порядок: сначала силлабус, затем блоки по темам согласно программе.

## Язык и программа

Материалы на русском и/или английском. Курс: BBA.
