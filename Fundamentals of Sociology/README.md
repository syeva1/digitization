# Fundamentals of Sociology

Курс Сколково: основы социологии.

## Для кого и зачем

Курс даёт базовые концепции и методы социологии: классические тексты (Дюркгейм, Вебер, Гофман), введение в методы сбора и анализа данных, демография, социальные сети, интервью, рациональность. Подходит для тех, кто не изучал социологию ранее.

## Содержание

Темы: классическая социологическая теория (Дюркгейм — правила метода, самоубийство; Вебер — протестантская этика; Гофман — представление себя в повседневной жизни); методы (введение в методы, интервью, демография, тесты личности, рациональность, социальные сети); кейс по демографии.

## Как устроены материалы (модули по порядку)

- **Модуль 1. Силлабус** — папка `Силлабус/`: программа курса (Syllabus 2024). Начните с неё.
- **Модуль 2. Литература_классика** — тексты Дюркгейма, Вебера, Гофмана (правила метода, самоубийство, протестантская этика, представление себя). В README папки — зачем каждая книга и как помогает.
- **Модуль 3. Методы_и_методология** — введение в методы, интервью, демография, личностные тесты, рациональность, социальные сети, кейс по демографии.

Рекомендуемый порядок: Модуль 1 → Модуль 2 → Модуль 3.

## Язык и программа

Материалы на английском. Курс входит в программу BBA.
