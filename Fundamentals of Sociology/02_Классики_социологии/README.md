# Классические тексты социологии

Тема: Дюркгейм, Вебер, Гофман — основы социологической теории.

## Зачем эти книги (как они помогают в курсе)

- **Дюркгейм, «Правила социологического метода»** (Moscow_Durkheim_Rules) — основа для понимания, как социология видит «социальные факты» и как их наблюдать; читать в начале блока по теории.
- **Дюркгейм, «Самоубийство»** (Moscow_Durkheim_Suicide) — классическое исследование; показывает применение правил метода к конкретной теме; читать после «Правил метода».
- **Вебер, «Протестантская этика и дух капитализма»** (Moscow_Weber_P1, P2) — вводит понятия идеальных типов и связи культуры с экономикой; даёт основу для анализа ценностей и действий.
- **Гофман, «Представление себя в повседневной жизни»** (Goffman_Presentation_of_Self) — микросоциология, драматургический подход; помогает анализировать повседневное взаимодействие и «игру» ролей.

Блок даёт концептуальную основу для блока по методам; предшествует ему по порядку прохождения.

## Основные файлы

- Moscow_Durkheim_Rules.txt, Moscow_Durkheim_Suicide.txt — Дюркгейм.
- Moscow_Weber_P1.txt, Moscow_Weber_P2.txt — Вебер.
- Goffman_Presentation_of_Self_in_Everyday_Life.txt — Гофман.
