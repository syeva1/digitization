# Город как квест / Город как текст

Курс: город, архитектура, субкультуры, урбанистика.

## Для кого и зачем

Курс рассматривает город как текст и квест: архитектура (лекции 1–4), субкультуры, экономика и урбанистика, стили (гид Ксении Малич), структурная игра, материалы по Петербургу. Подходит для интереса к урбанистике и культуре города без обязательной предподготовки.

## Содержание

Темы: архитектура (лекции 1–2 и 3–4), субкультуры, экономика и урбанистика, гид по стилям, подготовительные материалы, игра Структурная игра, проект по Петербургу.

## Как устроены материалы

- **Силлабус** — программа курса «Город как текст» 2024 (final).
- **Архитектура_лекции_1_2**, **Архитектура_лекции_3_4** — лекции по архитектуре.
- **Лекции_архитектура_доп** — дополнительные материалы по архитектуре.
- **Лекции_Субкультуры_2024** — субкультуры.
- **Экономика_и_урбанистика** — лекция по экономике и урбанистике.
- **Гид по стилям - Ксения Малич** — гид по стилям.
- **Подготовительные_материалы** — файлы к подготовительному заданию.
- **Структурная_игра** — материалы игры по структуре.
- **Петербург NEW draft3_ред_CORR (2)** — материалы по Петербургу.

Рекомендуемый порядок: силлабус → Подготовительные_материалы → Архитектура (1–2, 3–4) → Субкультуры → Экономика_и_урбанистика → остальное по программе.

## Язык и программа

Материалы на русском. Курс входит в программу BBA.
