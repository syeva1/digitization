# Литература: философия техники и автоматизации

Тема: тексты по истории и философии техники, кибернетике, постчеловеку, автоматизации труда.

## О чём материалы

Подборка текстов: Галисон (Винер, кибернетическое видение), Жилль (история техники, т. 1–2), Кангийем (машина и организм), Хейлз (как мы стали постчеловеком), Маккензи (знающие машины), Бенанав (автоматизация и будущее труда), Симондон (технические объекты), Маркс, материалы по образу и медиа. Охватывают все темы курса.

## Связь с курсом

Основной блок для чтения; порядок тем задаётся силлабусом.

## Основные файлы

Каждый .txt в папке — один текст или книга; имена файлов отражают автора и краткое название.
