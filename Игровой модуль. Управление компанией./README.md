# Игровой модуль: Управление компанией

Курс: управление компанией в игровом формате, командная работа.

## Для кого и зачем

Модуль даёт опыт принятия управленческих решений в формате игры/симуляции; тема командной работы компании (Титарев, BBA3). Подходит для закрепления тем по менеджменту и работе в команде.

## Содержание

Темы: программа модуля (силлабус 2025–26), управление командной работой компании (материалы Титарева для BBA3).

## Как устроены материалы

- **Силлабус_Управление компанией_2025-26** — программа модуля.
- **Управление_командной_работой_компании_Титарев_BBA3** — материалы по командной работе.

Рекомендуемый порядок: силлабус, затем блок по командной работе.

## Язык и программа

Материалы на русском. Модуль: BBA3.
