# Политические исследования

Курс Сколково: политология, количественные методы, регрессия, власть, государство, общество.

## Для кого и зачем

Курс даёт систематическое представление о политической науке и количественных методах: власть, государство, общество, политические технологии, регрессия и инструментальные переменные (Стукал), качественные методы. Преподаватели: Кирилл Телин, Денис Стукал. Подходит для изучения политологии и прикладных методов в политических исследованиях.

## Содержание

Темы по программе: власть (Вебер, Лукс, Вебер); государство (Джессоп, Скотт, Де Сото); общество; политические технологии; количественные методы (регрессия, дамми-переменные, инструментальные переменные, эндогенность). Литература: Атнашев и др. (несовершенная публичная сфера), Хархордин, Скотт «Благими намерениями государства», Джессоп «Государство», Сморгунов (государственная политика и управление), Шмитт (политология) и др.

## Как устроены материалы

- **Силлабус** — программа курса «Политические исследования» 2024 (файл может быть в кодировке UTF-8 после конвертации из Windows-1251).
- **Презентации_регрессия_Стукал** — презентации Д. Стукала: регрессия (skolkovo1), дамми и регрессия (skolkovo_reg_demo, skolkovo_dummy_demo), эндогенность и инструменты (skolkovo2_endog).
- **Литература_политические_исследования** — книги и статьи по курсу (в т.ч. по темам 21.10.24).
- В корне: полные тексты лекций (2024_Сколково_день_второй_полный_текст, Сколково_день3_FULL_1to1, Сколково_день4_FULL_1to1), тексты книг (Скотт, Джессоп, Сморгунов, Шмитт) и оцифрованные лекции (skolkovo1_FULL_1to1_v2).

Рекомендуемый порядок: силлабус → блок «Власть/Государство/Общество» (по лекциям день 2–4) → блок «Количественные методы» (Презентации_регрессия_Стукал и skolkovo1) → Литература по темам.
