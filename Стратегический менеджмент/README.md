# Стратегический менеджмент

Курс Сколково (Курс_СМ): стратегическое мышление, миссия и видение, разработка бизнес-стратегий, MECE, пирамида, детализация, сообщение, консультанты, корпоративные стратегии.

## Для кого и зачем

Курс даёт основы стратегического менеджмента: введение в стратегическое мышление, миссия/видение/ценности, этапы разработки бизнес-стратегий (диагностика, стратегический этап, детализация), принцип MECE и принцип пирамиды, стратегическое сообщение и личные стратегии, приглашение внешних консультантов, корпоративные стратегии. Подходит для BBA и практики стратегического планирования.

## Содержание

Темы задаются папками: Тема_1 (введение), Тема_2 (миссия, видение, ценности), Тема_3 части 1–5 (разработка бизнес-стратегий, этап диагностики), Тема_4 (этап стратегических решений), Тема_5 (MECE, пирамида), Тема_6 (детализация), Тема_7 (сообщение и личные стратегии), Тема_9 (консультанты), Тема_10 (корпоративные стратегии); организационная информация; силлабус.

## Как устроены материалы

- **Силлабус_Стратегический_менеджмент_2025_26** — программа курса.
- **Курс_СМ_Организационная_информация** — организационная информация.
- **Курс_СМ_Тема_1** … **Курс_СМ_Тема_10** — темы курса (номера и названия в именах папок).

Рекомендуемый порядок: силлабус → Организационная_информация → Тема_1 → Тема_2 → Тема_3 (части 1–5) → Тема_4–7, 9, 10 по программе.

## Язык и программа

Материалы на русском. Курс: BBA.
