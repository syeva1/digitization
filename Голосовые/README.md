# Голосовые

Транскрипты голосовых сообщений с лекций по проектному менеджменту и стратегическому менеджменту (курсы Сколково). В папке остаются только исходные расшифровки по отдельным записям; объединённые файлы и выжимка перенесены в папки курсов.

## Файлы в этой папке

- **voice_*_transcription.txt** — исходные транскрипты отдельных голосовых (13 файлов по проектному менеджменту).

## Куда перенесено

- **Проектный менеджмент:** [Проектный_менеджмент_голосовые_объединённо.txt](../Проектный%20менеджмент/Проектный_менеджмент_голосовые_объединённо.txt), [Выжимка_голосовых_темы_уроков.md](../Проектный%20менеджмент/Выжимка_голосовых_темы_уроков.md).
- **Стратегический менеджмент:** [Стратегический_менеджмент_голосовые_объединённо.txt](../Стратегический%20менеджмент/Стратегический_менеджмент_голосовые_объединённо.txt) (заготовка под будущие транскрипты).
