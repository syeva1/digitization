# Агенты проекта

Инструкции для Cursor Agent: все правила лежат в `.cursor/rules/` (симлинки на `Agents/*/`). Вызов в чате: **`@`** и имя правила.

## Как вызвать

В чате Cursor наберите **`@`** и начните вводить имя агента — появится список. Выберите нужный агент, затем опишите задачу.
