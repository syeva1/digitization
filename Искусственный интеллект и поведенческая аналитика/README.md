# Искусственный интеллект и поведенческая аналитика

Курс Сколково: ИИ, поведенческие данные, когнитивные искажения (AIBA).

## Для кого и зачем

Курс связывает искусственный интеллект и поведенческую аналитику: анализ поведенческих данных (Buisson и др.), когнитивные искажения (Wilke, Mata), применение в бизнесе. Подходит для интереса к данным и поведенческой экономике.

## Содержание

Темы задаются силлабусом (Syllabus AIBA Nov2025); блоки ридингов по темам (блоки 2, 3, 6, 7), общие материалы по сегментации, когнитивным искажениям.

## Как устроены материалы

- В корне: **Syllabus AIBA Nov2025 (1).txt** — программа курса; тексты Buisson (Segmentation), Wilke-Mata (Cognitive Bias).
- **Ридинг_блок_2**, **Ридинг_блок_2_вариант_A** — пре-ридинги ко второму блоку занятий.
- **Ридинг_блок_3**, **Ридинг_блок_6**, **Ридинг_блок_7** — пре-ридинги к блокам 3, 6, 7.

Рекомендуемый порядок: силлабус, затем ридинги по номерам блоков (2 → 3 → 6 → 7).
