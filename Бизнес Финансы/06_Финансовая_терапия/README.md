# Тема: Финансовая терапия

Тема блока: психологические и поведенческие аспекты финансовых решений (финансовая терапия).

## О чём материалы

Материалы по финансовой терапии: связь финансового поведения с установками и практиками; применение в контексте курса.

## Связь с курсом

Тематический блок в рамках курса по бизнес-финансам для бакалавров.
