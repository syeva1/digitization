# Бизнес Финансы / Корпоративное управление

Курсы (BBA): корпоративное управление, совет директоров, ESG, риски, финансовая терапия.

## Для кого и зачем

Материалы охватывают корпоративное управление (введение, совет директоров), ESG-трансформацию, управление рисками, исследование советов директоров, финансовую модель и рефлексию. Подходят для студентов BBA, интересующихся корпоративными финансами и управлением.

## Содержание

Темы: введение в корпоративное управление; совет директоров; ESG; управление рисками; исследование СД; презентации и методики (шкала состояний, финансовая терапия); шаблон финансовой модели; утренняя рефлексия для бакалавров.

## Как устроены материалы

- **Силлабус_корпоративное управление_BBA_Е Дубовицкая** — программа курса по корпоративному управлению (Дубовицкая).
- **Силлабус ВВА - Evgeny Plaksenkov** — программа по бизнес-финансам (Плаксенков).
- **BBA_Тема_1_Введение_в_корпоративное_управление** — вводный блок.
- **Тема 2. Совет директоров** — материалы по совету директоров.
- **ESG_трансформация_бакалавриат_16_12_final** — блок по ESG.
- **Управление_рисками** — управление рисками.
- **Исследование_СД_2025_v1-10 FINAL** — исследование советов директоров.
- **Бакалавры_Презентация_Евгений_Плаксенков_2025** — презентация Плаксенкова.
